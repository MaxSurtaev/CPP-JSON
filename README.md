***JSON***
*A C++ JSON library*
Extremely simple JSON C++ library.
- No dependency, just STL
- No BOOST
- fast and simple
- C++11

*For what*
- config files
- structure objects and serialisation them

*everything you need*
- load and dump serialisation
- operator[]
- foreach loop
- move semantic
